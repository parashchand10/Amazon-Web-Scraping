# Amazon Web Scraping using Python

## Project Overview

This project demonstrates how to extract product data from Amazon using Python and BeautifulSoup.

The script automatically collects:

- Product Name
- Price
- Ratings
- Reviews

No manual data collection required — everything is automated using Python.

## Technologies Used

- Python – Main programming language used to build the web scraping script.
- Requests – Sends HTTP requests to fetch webpage content from Amazon.
- BeautifulSoup – Parses HTML and extracts required data from the webpage.
- LXML – Fast and efficient parser used to process HTML structure.
- CSV – Stores the extracted data in a structured file format for analysis.

## How It Works

- Send request to Amazon webpage
- Parse HTML using BeautifulSoup
- Extract required product details
- Store data in CSV file

## Output

Structured product data ready for: 
- Data Analysis
- Price Tracking
- Market Research
- Automation Projects
